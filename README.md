# Monster Type Game

A small typing game built with React and Vite. You type a magic word before the monster reaches you. 

**[Motion](https://motion.dev)** (`motion/react`) handles the UI animations (monster approach, spell flash, wizard feedback, letter micro-interactions).

<img src="https://raw.githubusercontent.com/felipedfe/felipedfe/refs/heads/main/assets/type-monster-frame-2.png" alt="" />

## Run locally

```bash
npm install
npm run dev
```
